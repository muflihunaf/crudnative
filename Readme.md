file index.php halaman awal website
file db/koneksi.php adalah file koneksi ke database
file db/fungsi.php adalah file berisikan function SQL
folder asset adalah kumpulan file css
file db/cari.php adalah file berisikan searching / pencarian
file edit.php adalah file berisikan edit data barang
file excel_reader.php adalah libary untuk melakukan import excel
file export.php adalah file yang berfungsi untuk meng-ekspor data dalam bentuk excel
file hapus.php adalah file yang berfungsi untuk hapus data barang dan juga berisi code trigger
file lihat_cadangan.php adalah file yang memiiki fungsi menampilkan data barang yang sudah dihapus
file lihat_barang.php adalah file yang memiliki fungsi menampilkan data barang yang tersedia
file tambah_barang.php adalah file yang berfungsi untuk menambahkan data barang kedalam database
file upload.php adalah file yang berfungsi menampilkan form untuk import data
file upload_aksi.php adalah file yang berfungsi untuk melakukan import kedalam database
